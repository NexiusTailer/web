## توضیحات

زاویه رو به رو بودن NPC رو تنظیم می‌کنه.

| نام                 | توضیح                   |
| -------------------- | ----------------------------- |
| Float:ang            | زاویه جدید رو به رو بودن NPC.   |

## مثال‌ها

```c
SetMyFacingAngle(0); //NPC roo be shamal mikone
```
