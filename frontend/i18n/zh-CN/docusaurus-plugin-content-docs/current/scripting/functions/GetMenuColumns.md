## 描述

获取菜单中当前激活的列数。

| 参数名      | 说明                |
| ----------- | ------------------- |
| Menu:menuid | 要获取列数的菜单 ID |

## 返回值

返回激活的列数。

## 示例代码

```c
new columns = GetMenuColumns(menuid);  // 获取激活的列数
```
