## 描述

该函数可作为 GetTickCount 的替代方案，返回服务器启动至今经过的毫秒数。

| 名称           | 描述                                                                             |
| -------------- | -------------------------------------------------------------------------------- |
| &granularity=0 | 该引用参数返回时将被赋值为系统内部时钟每秒的滴答数，用于指示函数返回值的时间精度 |

## 返回值

系统启动后经过的毫秒数。对于 32 位存储单元，该计数器在连续运行约 24 天后会溢出。

## 相关函数

- [GetTickCount](GetTickCount): 返回一个每毫秒递增的时间戳数值
