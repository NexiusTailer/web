## 描述

获取玩家 ID 的默认颜色。

| 参数名   | 说明                                |
| -------- | ----------------------------------- |
| playerid | 要获取颜色的玩家 ID（无需玩家在线） |

## 返回值

返回玩家的默认颜色值。

## 示例代码

```c
new colour = GetDefaultPlayerColour(playerid);  // 获取玩家的默认颜色
```
